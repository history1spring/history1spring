점수 계산

  • 최종 점수 = 카드 점수 합 + 남은 열쇠
  • 열쇠는 1개당 1점
  • 금화는 점수 아님, 동점일 때만 비교
  • 뒷면 카드는 점수와 아이콘 없음

자물쇠 능력

  • 자물쇠 카드 위에 열쇠 1개
  • 능력 사용 시 그 열쇠 제거
  • 일회용, 원하는 타이밍에 사용
  • 한 차례에 자물쇠 능력 1개까지만 사용

확장 준비

  • 기본 78장 + 확장 12장
  • 확장 구성: 성 6장, 마을 6장
  • 앱 시작 데이터: 총 90장
  • 확장 카드는 자물쇠/확장 배지 표시

문장 아이콘

귀족 신앙 학문 군사 공예 농민

자주 보는 기호

금화: 동점 비교용, 점수 아님
열쇠: 남은 열쇠 1개당 1점
자물쇠: 카드 위 열쇠 제거 후 일회성 능력 사용
뒷면 카드: 점수, 방패, 배너, 능력 없음
